## 📌 Overview

The **Advanced Scientific Calculator** is a console-based application written in the C programming language.
It is designed to perform both basic and scientific mathematical operations using a clean, modular, and menu-driven interface.

This project demonstrates strong fundamentals of C programming, including functions, control flow, and use of standard libraries.

---

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖ Multiplication
* ➗ Division (with divide-by-zero protection)
* 🧮 Modulus
* 🔼 Power Calculation
* √ Square Root
* 🔁 Continuous operation until exit
* ⚠ Error handling for invalid operations

